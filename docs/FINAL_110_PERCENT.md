## 🚀 快速开始 (30秒启动)

### 方式1: 一键启动 (推荐)
```bash
# 克隆仓库
git clone https://github.com/kunlunxingqiong/ruoxi-v2.git
cd ruoxi-v2

# 一键启动
python scripts/quick_start.py
```

### 方式2: 分步启动
```bash
# 1. 检查环境
python scripts/check_env.py

# 2. 配置环境
cp .env.example .env
# 编辑 .env 填入API密钥

# 3. 启动服务
python scripts/quick_start.py

# 4. 测试API
python scripts/test_api.py
```

### 方式3: Docker
```bash
docker-compose up -d
```
