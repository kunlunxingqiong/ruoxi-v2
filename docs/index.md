# 若曦V2 文档中心

欢迎来到若曦V2的文档中心。

## 项目概述

若曦V2是一个智能少女AI交互系统，专注于提供真实、温暖、有情感的人机交互体验。

### 核心特性

- 🌸 **情感渲染**: 根据情绪状态生成富文本表现
- 🧬 **自适应语言**: 根据文档类型调整语言风格
- 💫 **边缘时刻处理**: 优雅处理对话中的沉默、返回等边缘场景
- 🔒 **企业级安全**: 完整的安全审计和测试覆盖

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行服务

```bash
python -m platform.backend.main
```
