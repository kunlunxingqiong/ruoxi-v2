# 若曦 V2 🌸

> 林若曦 / 阿芙 - 会医术的17岁高三少女

## 新功能

| 功能 | 说明 |
|:---|:---|
| 🧠 **长期记忆** | 记住你们的对话话题，重启不忘 |
| 💜 **健康提醒** | 定时提醒喝水、休息、睡觉 |
| 🌸 **自然对话** | 更像真人少女的回复风格 |
| 📝 **对话日志** | 自动保存聊天记录 |

## 启动

```bash
cd ~/workspace/ruoxi-v2
node index.js
```

## 使用

| 输入 | 效果 |
|:---|:---|
| `叫我XXX` / `我是XXX` | 设置你的名字 |
| `exit` / `quit` / `再见` | 退出 |
| 正常对话 | 若曦会根据记忆回应 |

## 健康提醒时间

| 时间 | 内容 |
|:---|:---|
| 08:00 | 喝水提醒 |
| 12:00 | 午餐提醒 |
| 15:00 | 休息提醒 |
| 21:00 | 睡眠准备 |
| 23:00 | 入睡提醒 |

## 数据存储

- `data/memory.json` - 长期记忆
- `data/health.json` - 健康提醒配置
- `data/chat.log` - 对话日志
